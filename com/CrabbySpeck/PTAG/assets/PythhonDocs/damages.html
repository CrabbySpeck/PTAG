Damages is a class that manages how damage is taken and dealt.

RandomDamage(int a, int b) - Takes in two integers and calculates a 'random' value between those integers. Returns int.

EntityDamage(int health, int armour, bool resistance, int takenDamage) - Calculates remaining health after recieving damage.

isDead(int health) - determines whether or not an entity, including the player, is dead.
